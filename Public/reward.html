2015年在线课堂课程奖励规则

一、在线课堂课时奖励

单次课堂在线学员人数达到10人，将享有100元课程费用。2015年2月1日起，实行以下阶梯式学员奖励机制：

1. 课堂在线学员人数在10人以下，无课堂奖金；

2. 课堂在线学员人数为10人，课程费用100元；

3. 课堂在线学员人数达到（11-20）人之间，每增加一名学员，可获得10元学员额外课程费用；

4. 课堂在线学员人数达到（21-30）人之间，每增加一名学员，可获得15元学员额外课程费用；

5. 课堂在线学员人数达到31人（含）以上，不再累计课程费用；

假定一次课堂在线学员人数为N:

在线学员人数 课堂奖励 额外课程费用 课程费用
n＜10 0 0 0
n=10 100 0 100
10＜N≤20 100 （N-10）×10 100+[（N-10）×10]
20＜N≤30 100 （N-20）×15+10×10 100+[10×10+（N-20）×15]
N＞30 100 10×10+10×15 350

二、教师课时上限

2015年5月1日起，原则上普通教师每个月开课数量不超过25节。

三、在线课堂回放课奖励

1. 管理员关闭回放课不计入授课奖励范围。

2. 教师关闭回放课不计入授课奖励范围。

3. 无回放课状态不计入授课奖励范围。

4. 回放课状态教师可操作时间为回放课生成后24小时内。

四、在线课堂质量基本要求

在线课堂的课程要满足以下基本要求。

（一）在线课堂授课基本要求

1. 要求在线课上课前调整好麦克风和摄像头，麦克风有声音，摄像头有图像，上传的PPT课件或其他格式课件播放正常，要求教师需要讲课。

2. 要求麦克风发出的声音不得是与本课内容无关的话题。

3. 要求摄像头拍摄的画面中必须出现老师头像。

4. 要求第一次开课就是测试课，要带测试两字，测试一切顺利，指的是：麦克风有声音，摄像头有图像，上传的PPT或其他格式课件播放正常。（正式开课可不带测试两字）

（二）回放课展示基本要求

1. 要求麦克风有声音，摄像头有图像，上传的PPT或其他格式课件播放正常。

（三）在线课堂授课教师规范要求

1. 要求教师教态自然大方，亲和力好。

2. 要求教学语言规范，语速适中，吐字清楚，合理运用肢体语言。

3. 要求教学设计适合学习者特征（诸如学生年龄、文化背景、汉语水平），遵守有效、省时、目的性强。

4. 要求教学工具（例如PPT、音视频、图片、卡片等材料，画笔、荧光笔、投票等本网站课堂功能）设计和使用符合课堂内容，有效辅助课堂教学。

5.教师使用学生母语（例如英语、日语、法语、俄语等）进行教学，比例适当，运用时机恰当。

6. 要求有良好的教室规则，教室背景整洁干净。

7. 要求讲授生动，启发思维，教学理念新颖，教学目的明确，教学环节清晰，教学方法及教学手段独到，重点突出。
